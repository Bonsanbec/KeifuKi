# Keifu
Política de Privacidad

1. Responsable del tratamiento de datos
La aplicación y sus servicios son gestionados exclusivamente por el usuario propietario del dispositivo. No se recopilan datos por terceros ni se transmiten a servidores externos salvo que el usuario decida activar un respaldo personal.

2. Datos recopilados
La aplicación almacena localmente:
	•	Respuestas a preguntas en formato texto, audio, foto o video.
	•	Fecha y hora de cada respuesta.
	•	Identificador de la pregunta asociada.

No se recopilan datos personales adicionales, ubicación, contactos ni información de uso fuera del contexto de las respuestas.

3. Uso de los datos
Los datos se utilizan únicamente para:
	•	Mostrar la pregunta y su respuesta correspondiente.
	•	Permitir la revisión y visualización de entradas previas.
	•	Mantener estadísticas internas de uso de preguntas para la lógica de selección.

4. Compartición de datos
No se comparte ningún dato con terceros. Todos los archivos permanecen en el dispositivo del usuario o en medios de respaldo que éste configure explícitamente.

5. Respaldo y sincronización
Si el usuario decide utilizar almacenamiento en la nube personal (por ejemplo, Google Drive), los datos pueden ser copiados para su seguridad. Esta acción es voluntaria y bajo control exclusivo del usuario.

6. Seguridad
Los datos se almacenan localmente y se recomienda al usuario proteger su dispositivo con contraseñas o cifrado. La aplicación no transmite datos de forma automática a servidores externos.

7. Modificación y eliminación
El usuario puede eliminar cualquier respuesta en cualquier momento desde la interfaz de la aplicación, borrando así la información correspondiente de manera irreversible.

8. Cambios en la política
Esta política puede actualizarse para reflejar cambios en la funcionalidad de la aplicación o mejoras de seguridad. Los cambios no afectan retroactivamente la información ya almacenada.

9. Contacto
Toda gestión relacionada con la privacidad es controlada localmente por el usuario. La aplicación no mantiene contacto ni soporte externo para el manejo de datos.
